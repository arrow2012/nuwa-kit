# Nuwa Common Library

Shared utilities for Nuwa IAM and related services.

## Packages
- `log`: Structured logging (Zap wrapper)
- `json`: High-performance JSON (goccy/go-json wrapper)
- `options`: Standard configuration structs (Redis, MySQL, etc.)
- `idutil`: ID generation (Snowflake, Feistel)

## Usage
```bash
go get github.com/arrow2012/nuwa-common
```
